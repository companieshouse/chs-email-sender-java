# chs-email-sender-java
Request an email to be sent based on kafka messages

## Getting started

To checkout and build the service:

1. Clone [Docker CHS Development](https://github.com/companieshouse/docker-chs-development) and follow the steps in the
   README.
2. Run `./bin/chs-dev services enable chs-email-sender-java`
3. Run `./bin/chs-dev development enable chs-email-sender-java` if you wish to see changes in the code dynamically
4. Run `chs-dev up` in the docker-chs-development directory.

These instructions are for a local docker environment.
